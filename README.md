# vuex_crash_course

In this crash course you will learn all about Vue.js including what it is, Vue-CLI, components, data, events ...

![](images/vuex.png)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
